# КЕЙС
Отзывы горожан — ценная информация, которая помогает столице развивать городские цифровые сервисы. Но анализировать вручную большой объем комментариев — трудоемкая и практически невыполнимая задача для масштабов такого мегаполиса, как Москва. В связи с этим актуальна разработка и внедрение автоматизированных методов анализа тональности текста.
В рамках данной задачи требуется разработать модель машинного обучения, способную классифицировать тональность русскоязычных текстов, а также создать веб-приложение с интерфейсом для взаимодействия с моделью. Такой сервис поможет городу быстрее классифицировать обращения и оперативнее узнавать, что в цифровых проектах столицы нравится горожанам, а что требует улучшения.

Стек технологий:  
Python (любые библиотеки и ML/DS фреймворки)
Для frontend- и backend-части можете использовать любые доступные инструменты.
