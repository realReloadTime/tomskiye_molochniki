Для анализа с файлами идет запрос на:
POST http://localhost:5039/api/Analysis/analyze-file  

Мне нужно передавать файл в бинарном виде через multipart/form-data

Ответка:
{
    "analysisDate": "2024-01-15T10:30:00Z",
    "totalRecords": 150,
    "positiveCount": 120,
    "negativeCount": 30
}

Обязательная структура scv файла(это можешь просто где нибудь написать на странице)
Формат файла - только CSV с колонкой comment
Одна колонка с названием comment
Заголовок обязательно должен быть comment (первая строка)
Каждый комментарий на новой строке
Кодировка - UTF-8
Формат - обычный CSV (разделитель - запятая)
